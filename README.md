![HR](https://img.shields.io/badge/HR-Automation-blue)

An AI-powered HR Onboarding Automation platform that streamlines the employee onboarding process by automating document collection, employee registration, task assignments, orientation scheduling, email notifications, and onboarding progress tracking.

# 📖 Overview

HR Onboarding Automation helps HR teams reduce repetitive manual work and improve employee onboarding experiences. The platform centralizes onboarding workflows, making it easier to manage employees from offer acceptance to successful onboarding completion.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/hr-onboarding-automation.git
```

Move into project

```bash
cd hr-onboarding-automation
```

Install dependencies

```bash
npm install
```

Start backend

```bash
npm run server
```

Start frontend

```bash
npm start
```

# 💻 Usage

1. HR Admin logs in.
2. Add a new employee.
3. Upload onboarding documents.
4. Assign onboarding tasks.
5. Employee receives welcome email.
6. Employee uploads required documents.
7. HR verifies documents.
8. Manager approves onboarding.
9. Employee onboarding is completed.
